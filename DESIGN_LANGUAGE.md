# Design Language

Spelunk should feel like premium engineering software for scientific inspection.

## Tone

- precise
- calm
- instrument-like
- evidence-oriented
- never gimmicky

The cave metaphor may appear in language, startup treatment, subtle empty states, and small ASCII motifs. It must not interfere with research workflows.

## Visual Rules

- Every screen answers one question.
- Put conclusions before charts.
- Use progressive disclosure.
- Use semantic color only.
- Preserve whitespace.
- Keep animation subtle and meaningful.
- Avoid scrolling logs in normal workflows.

## Terminal Interaction

Keyboard-first:

- arrows move selection
- enter opens
- escape goes back
- tab cycles panes
- ctrl+p opens command palette
- `/` searches
- `?` opens shortcuts
- `q` quits

Mouse support is useful but secondary.
